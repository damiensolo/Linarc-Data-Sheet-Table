# Keyboard Shortcuts Reference

This document is for using Spreadsheet+, Advanced Spreadsheet, and shared app screens.

## Key Names

- **Mac:** `Cmd` means Command.
- **Windows/PC:** `Ctrl` means Control.

---

## App-Level Shortcuts

### Open search
- **Mac:** `/`
- **PC:** `/`
- **Scope:** View controls / global search bar

### Quick Create menu
- **Mac:** No keyboard shortcut (mouse only)
- **PC:** No keyboard shortcut (mouse only)
- **Scope:** Main navigation Quick Create

### Close active menu or modal
- **Mac:** `Esc`
- **PC:** `Esc`
- **Scope:** Menus and dialogs (for example, view menus and modal overlays)

### Confirm create/rename inputs
- **Mac:** `Enter`
- **PC:** `Enter`
- **Scope:** Create/Rename style text inputs

---

## Spreadsheet+ (V3) and Advanced Spreadsheet (V4)

These shortcuts are implemented in both spreadsheet variants.

### Navigate focused cell
- **Mac:** Arrow keys
- **PC:** Arrow keys
- **Behavior:** Move focus up/down/left/right by one cell

### Move to next/previous cell
- **Mac:** `Tab` / `Shift + Tab`
- **PC:** `Tab` / `Shift + Tab`
- **Behavior:** Move right/left; wraps to next/previous row at boundaries

### Enter edit mode
- **Mac:** `Enter`
- **PC:** `Enter`
- **Behavior:** Start editing current editable cell

### Clear current cell value
- **Mac:** `Delete` or `Backspace`
- **PC:** `Delete` or `Backspace`
- **Behavior:** Clears focused editable cell

### Start typing to edit
- **Mac:** Type any printable key
- **PC:** Type any printable key
- **Behavior:** Starts editing and seeds value with typed character

### Clipboard (cell level)
- **Mac:** `Cmd + C`, `Cmd + X`, `Cmd + V`
- **PC:** `Ctrl + C`, `Ctrl + X`, `Ctrl + V`
- **Behavior:** Copy, cut, paste for active cell context

### Undo / Redo
- **Mac:** `Cmd + Z`, `Cmd + Shift + Z`, `Cmd + Y`
- **PC:** `Ctrl + Z`, `Ctrl + Shift + Z`, `Ctrl + Y`
- **Behavior:** Undo or redo recent spreadsheet edits

### Indent row
- **Mac:** `Cmd + ]`
- **PC:** `Ctrl + ]`
- **Behavior:** Indents selected/current row under previous sibling (when valid)

### Outdent row
- **Mac:** `Cmd + [`
- **PC:** `Ctrl + [`
- **Behavior:** Outdents selected/current row one level up (when valid)

### Expand / collapse hierarchy
- **Mac:** `Alt + Right Arrow` / `Alt + Left Arrow`
- **PC:** `Alt + Right Arrow` / `Alt + Left Arrow`
- **Behavior:** Expands or collapses the focused row and its expandable descendants

### Toggle row selection
- **Mac:** `Shift + Space`
- **PC:** `Shift + Space`
- **Behavior:** Selects/unselects the focused row

### Edit focused cell
- **Mac:** `F2`
- **PC:** `F2`
- **Behavior:** Opens the focused editable cell in edit mode

## Context Menu Actions (Mouse + Menu)

The spreadsheet row/cell context menu also exposes these actions:

- Insert row above
- Insert row below
- Add child row
- Indent row
- Outdent row
- Clear cell
- Delete row

Some entries show shortcut hints directly in the menu (for example, Copy/Cut/Paste and Indent/Outdent).

---

## Notes

- Shortcuts are context-sensitive. A spreadsheet shortcut works only when the spreadsheet has focus.
- Row hierarchy shortcuts (`Cmd/Ctrl + ]` and `Cmd/Ctrl + [`) are supported in both Spreadsheet+ and Advanced Spreadsheet.
- If a shortcut appears not to fire, click a spreadsheet cell first, then retry.
